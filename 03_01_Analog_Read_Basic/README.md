# 03_01 - Analog Read Basic

Bài học này hướng dẫn cách đọc tín hiệu Analog từ một biến trở và in giá trị lên Serial Monitor.

## Chức năng
- Đọc giá trị từ 0 đến 1023 từ chân A0.
- Hiển thị giá trị thô lên màn hình máy tính.

## Kết nối
- Biến trở (Potentiometer):
    - Chân 1: GND.
    - Chân 2 (Chân giữa): A0.
    - Chân 3: 5V.

## Cách xem kết quả
Mở **Serial Monitor** trong Arduino IDE với tốc độ **9600 baud** để xem dữ liệu.
