# IoT Project Repository

Dự án này bao gồm các bài tập lập trình Arduino cơ bản để làm quen với các linh kiện điện tử như LED, Nút nhấn và Biến trở.

## Cấu trúc dự án

Dưới đây là danh sách các bài học có trong repository này:

| Folder | Mô tả | Linh kiện |
| :--- | :--- | :--- |
| [01_LED_Momentary](./01_LED_Momentary) | Bật/tắt LED tức thời theo nút nhấn. | LED, Nút nhấn |
| [02_LED_Toggle](./02_LED_Toggle) | Nhấn nút để đảo trạng thái LED. | LED, Nút nhấn |
| [03_01_Analog_Read_Basic](./03_01_Analog_Read_Basic) | Đọc tín hiệu Analog từ biến trở. | Biến trở (Potentiometer) |
| [03_02_LED_Toggle_Debounce](./03_02_LED_Toggle_Debounce) | Nút nhấn đảo trạng thái LED có chống rung phần mềm. | LED, Nút nhấn |
| [03_03_Potentiometer_Voltage](./03_03_Potentiometer_Voltage) | Quy đổi tín hiệu Analog sang điện áp (V). | Biến trở (Potentiometer) |

## Yêu cầu
- Phần cứng: Arduino Uno (hoặc tương đương).
- Phần mềm: Arduino IDE.
