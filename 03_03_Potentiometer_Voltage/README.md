# 03_03 - Potentiometer Voltage Measurement

Bài học này hướng dẫn cách chuyển đổi giá trị Analog đọc được sang giá trị điện áp (Voltage) thực tế.

## Công thức tính
Điện áp (V) = (Giá trị Analog * 5.0) / 1023.0

## Chức năng
- Đọc giá trị Analog từ biến trở.
- Tính toán và in ra giá trị điện áp (ví dụ: 2.50V, 5.00V) lên Serial Monitor.

## Kết nối
- Biến trở (Potentiometer): Chân giữa nối A0.
